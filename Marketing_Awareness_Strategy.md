## Senegal Smart Guide Hub - Marketing & Awareness Plan

## 📢 استراتيجية التسويق الشاملة

### **الهدف الرئيسي:**
جعل نظام "بصمة التوجيه السنغالية" **الحل الموثوق الأول والأخير** لأومان الحجاج والمعتمرين السنغاليين.

### **الشريحة المستهدفة الأولية:**
- 30,000 حاج سنغالي سنوياً
- 50,000 معتمر سنوياً
- 100,000 عائلة في الوطن (مصدر الطلب)
- الجهات الحكومية (وزارة الشؤون الدينية، الأمن...)

## 🎯 المراحل التسويقية

### **المرحلة 1: التوعية الحكومية (قبل البدء بـ 6 أشهر)**

#### الهدف:
الحصول على موافقة وتمويل حكومي والشراكات الدولية.

#### الاستراتيجيات:

**1. عروض تقديمية حكومية:**
```
📍 الجهات:
  - مجلس الوزراء (عرض رسمي)
  - لجنة الشؤون الدينية بالبرلمان
  - وزارة الداخلية والأمن
  - السفارة السنغالية بالسعودية

📊 المحتوى:
  - ملخص تنفيذي (15 دقيقة)
  - فيديو توضيحي (3 دقائق)
  - عرض شرائح احترافي
  - Q&A focused
```

**2. قمة وزارية مع السعودية:**
```
📍 الموقع: جدة (حيث سفارتنا)
📅 التوقيت: شهر رمضان (قريب من الحج)
👥 المشاركون:
  - وزير الشؤون الدينية السنغالي
  - مسؤول أمن حج سعودي
  - مديرو مشاريع تقنيين

📄 النتيجة المتوقعة:
  - توقيع MOU (Memorandum of Understanding)
  - تحديد آلية التنسيق الفنية
```

**3. جذب شراكات دولية:**
```
🤝 الاتصالات:
  - Microsoft / Google / AWS (لتقديم خوادم مجانية)
  - البنك الإسلامي للتنمية (تمويل)
  - UNDP (دعم فني)

💼 الحوافز:
  - فرصة لتحسين سمعتهم الدولية
  - عرض حالة دراسية (Case Study) جديدة
  - ترخيص تقني محتمل في المستقبل
```

### **المرحلة 2: التوعية الشعبية (قبل التجربة بـ 3 أشهر)**

#### الهدف:
بناء ثقة وفهم شامل بين الحجاج والمعتمرين.

#### الاستراتيجيات:

**1. حملة إعلانية تلفزيونية:**

```
📺 الاسم: "آمن في رحلتك"
🎬 الرسالة الرئيسية:
  "من الآن، لن تضيع أبداً في الحرم"

📽️ محتوى الفيديو (3 إصدارات):

[إصدار 1: قصة إنسانية]
  - شيخ كبير ضاع
  - أسرته قلقة جداً
  - ارتدى السوار
  - وجدوه في 10 دقائق
  - فرحة أسرته

[إصدار 2: توضيحي تقني]
  - كيف يعمل السوار
  - البيانات الآمنة
  - الاستجابة الفورية
  - رسالة الأمان

[إصدار 3: موجه للأسرة]
  - زوجة تراقب زوجها
  - تطمئن عليه
  - لماذا هذا مهم
  - كيف تطلب المساعدة
```

**2. حملة وسائل التواصل الاجتماعي:**

```
📱 Platforms:
  - Facebook (أكثر استخدام بين الحجاج الكبار)
  - WhatsApp (مجموعات حجاج)
  - TikTok (الشباب)
  - Twitter (قادة الرأي)

📝 محتوى يومي:
  ✓ قصص نجاح محاكاة
  ✓ نصائح الأمان
  ✓ أسئلة شائعة (FAQ)
  ✓ شهادات من الفريق التقني
  ✓ مسابقات وهدايا

🎁 المسابقة الشهيرة:
  "من الذي سيحظى بـ 100 سوار ذكي مجاني؟"
  - Hashtag: #بصمة_التوجيه
  - الفائزون: عائلات ستذهب للحج
```

**3. حملة إعادة (Retargeting):**

```
🎯 المسار:
  تصادف إعلان → تفكر → تبحث → تقرر

📊 الإعلانات:
  - Google Search Ads: "كيفية عدم الضياع في الحج"
  - YouTube Pre-roll: محتوى حج
  - Facebook Targeting: عمر 55+ ليسوا متعلمين تقياً

💬 الرسالة:
  "هزّ بساطة + أمان = حج هادئ"
```

### **المرحلة 3: التوعية المباشرة (قبل الموسم بـ شهر)**

#### الهدف:
تدريب الحجاج مباشرة على استخدام النظام.

#### الاستراتيجيات:

**1. ورش تدريبية في المساجد:**

```
🕌 أماكن التدريب:
  - المساجد الرئيسية (كل منطقة)
  - مكاتب التجمّع للحج
  - فنادق كبرى

⏰ الجدول:
  - 5 جلسات يومية
  - كل جلسة: 2 ساعة
  - تشمل عملي على السوار الفعلي

👨‍🏫 المدربون:
  - موظفو البعثة
  - متطوعون مدربون
  - شباب درسوا التقنية

📋 محتوى الورشة:
  1. مقدمة بـ الولوفية (20 دقيقة)
  2. كيفية ارتداء السوار (5 دقائق)
  3. كيفية فهم الأصوات والاهتزازات (10 دقائق)
  4. ماذا تفعل إذا ضعت (15 دقيقة)
  5. تطبيق عملي مع نموذج (30 دقيقة)
  6. أسئلة وأجوبة (10 دقائق)
  7. قرعة جوائز (10 دقائق)
```

**2. فيديوهات تدريبية قصيرة:**

```
📹 الفيديوهات (بـ الولوفية مع العربية):

[Video 1: "السوار الذكي - شرح بسيط" (2 دقيقة)]
- ماهو السوار؟
- لماذا تحتاج إليه؟
- كيف تلبسه؟

[Video 2: "الأصوات والتنبيهات" (1.5 دقيقة)]
- صوت الإنذار الأول
- صوت الإنذار الثاني
- الرسالة الصوتية الكاملة

[Video 3: "ماذا تفعل إذا ضعت" (2 دقيقة)]
- خطوات الأمان الأول
- كيفية طلب المساعدة
- أرقام الاتصال الطارئة

[Video 4: "قصة عم الحاج" (3 دقائق)]
- شيخ عمره 75 عام ضاع
- كيف ساعده السوار
- الدرس المستفاد
```

**3. منشورات توعية مطبوعة:**

```
📄 المنشoric الموزعة:

[منشور 1: دليل سريع]
- حجم: A5 (نصف ورقة)
- لغة: ولوفية + عربية + إنجليزية
- محتوى: رسوم توضيحية بسيطة
- توزيع: مع تصاريح الهج

[منشور 2: بطاقة أرقام الطوارئ]
- حجم: بطاقة (10 × 6 سم)
- تصميم: سهل الحفظ
- الأرقام: سفارة، شرطة، مستشفى
- تثبيت: على السوار نفسه

[منشور 3: شهادة المشاركة]
- بعد التدريب
- تصميم رسمي
- توقيع مسؤول
- حافز نفسي للالتزام
```

## 🏆 حملات الحوافز والجوائز

### **المسابقات الشهرية:**

```
🎁 الجائزة الأولى (للحجاج):
"من الذي سيحقق أقل وقت جود بدون ضياع؟"
- الجائزة: 1 مليون فرنك (عمرة مجانية السنة القادمة)
- القياس: من بيانات النظام
- التوقيت: نهاية كل موسم

🏅 الجائزة الثانية (للمتطوعين):
"أفضل متطوع شهر"
- المعايير: عدد الحالات، رضا المستخدمين، الالتزام
- الجائزة: شهادة رسمية + مكافأة مالية (100 ألف فرنك)
- الاحتفالية: حفل نهاية الموسم

🎖️ الجائزة الثالثة (للفريق):
"فريق عمليات سنغال الأفضل"
- الحكم: وزير الشؤون الدينية
- الجائزة: رحلة قصيرة مدفوعة + شهادة فخرية
```
