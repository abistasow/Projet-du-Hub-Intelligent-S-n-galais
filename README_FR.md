📌 APERÇU RAPIDE DU PROJET

Le Problème:
────────────
• Des milliers de pèlerins et fidèles sénégalais se perdent chaque année aux Lieux Saints
• Les opérations de recherche peuvent prendre des JOURS au lieu d'heures
• Barrière linguistique: les services ne sont qu'en arabe, mais nos pèlerins parlent wolof
• Les personnes âgées ne peuvent pas utiliser les applications mobiles complexes
• Aucun système de communication unifié entre l'ambassade et les autorités saoudiennes

La Solution:
────────────
Système technologique hybride intelligent qui comprend:
✓ Bracelet intelligent simple (NFC + QR Code)
✓ Système d'alertes audio géolocalisées en Wolof
✓ Application pour bénévoles (pas pour les pèlerins)
✓ Tableau de bord central pour le gouvernement
✓ Intégration complète avec les systèmes saoudiens
